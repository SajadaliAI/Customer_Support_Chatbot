# AI Customer Support Chatbot 🤖

This project is an **AI-based Customer Support Chatbot** built using **Python, Flask, and NLP techniques**.
The chatbot can answer common customer support questions by matching user queries with a dataset of frequently asked questions.

## 📌 Features

* NLP-based question matching using **TF-IDF Vectorizer**
* **Cosine Similarity** for finding the most relevant response
* Web interface using **Flask**
* FAQ dataset-based response system
* Fallback response for unknown queries
* Clean and simple chatbot interface

## 🛠 Technologies Used

* Python
* Flask
* Pandas
* Scikit-learn
* Natural Language Processing (NLP)
* HTML / CSS / JavaScript
